## 📋 漫剧制作核心要点（从知识库学习）

### 1. 好的漫剧公式
```
爆款内容 + AI生成画面 + 动态转场 + 热门配音 = 爆款漫剧
```

### 2. 关键要素
- **选题**：蹭热点、情感共鸣、悬念冲突
- **脚本**：短平快、每3秒一个反转
- **画面**：风格统一、角色一致
- **节奏**：BGM卡点、字幕同步

## 📝 漫剧脚本模板

```

【类型】：情感/搞笑/悬疑/正能量
【时长】：30-60秒

【第1镜】：场景描述 - 对话/动作
【第2镜】：场景描述 - 对话/动作
...
【第n镜】：场景描述 - 对话/动作

【结尾引导】：关注我们/点赞收藏
```

1. 8-10个镜头
2. 每镜3-5秒
3. 有反转/冲突/情感共鸣
4. 适合AI绘图表现
5. 结尾有互动引导

输出格式：
【标题】：
【类型】：
【第1镜】：画面描述 - 台词
