## SKILL.md

Generate complete, publish-ready content for Chinese social media platforms.

## When to Activate

Trigger when the user mentions: 写文章, 写笔记, generate content, 写稿, draft post, create article.

## Workflow

- Confirm inputs:
  - Topic / title
  - Target platform
  - Word count
  - Style (professional / casual / product review / educational)
  - Whether image suggestions are needed

- Generate platform-native content

## Quality Checklist
- Is the title compelling?
- Does it hook the reader?
- Are there concrete examples?
- Is there a clear CTA?

## Guidelines
- All content must be original
- Avoid absolute claims
- Data citations must be verifiable
## SKILL.md

生成可用于发布的自媒体平台内容。

## 何时激活

当用户提到以下内容时触发：写文章、写笔记、生成内容、写稿、创建文章。

## 工作流程

- 确认输入：
  - 主题/标题
  - 目标平台
  - 字数要求
  - 风格（专业/休闲/产品评测/教育）
  - 是否需要图片建议

- 生成平台原生内容

### 小红书格式
- 标题带 emoji 和关键词
- 开头钩子吸引注意
- 短段落，每段 2-3 句
- 总结/行动号召
- 标签 5-10 个

### 知乎格式
- 直接回答核心问题
- 逻辑展开论证
- 引用数据或案例
- 要点总结
- 引导关注/点赞

### 公众号格式
- 标题
- 开头建立共鸣
- 多个子标题分段
- 结尾提升 + CTA

## 质量检查清单
- 标题是否吸引人？
- 前 3 行能否抓住读者？
- 是否有具体例子或数据？
- 是否有明确的行动号召？

## 指南
- 所有内容必须原创
- 避免绝对化表述
- 数据引用必须可验证
